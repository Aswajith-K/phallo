# Security Policy

## Supported Versions

We actively maintain the latest stable release of Phallo. Security updates and patches are primarily applied to the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | Yes (Current Active) |
| < 1.0.0 | No                 |

---

## Reporting a Vulnerability

> [!IMPORTANT]
> Please do not report security vulnerabilities via public GitHub issues. Instead, help us keep Phallo safe by reporting security bugs privately.

If you find a security vulnerability, open a private discussion or contact through GitHub.

Thank you for helping keep this project secure!
