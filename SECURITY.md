# Security Policy

## Supported Versions

We actively maintain the latest stable release of Phallo. Security updates and patches are primarily applied to the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | Yes (Current Active) |
| < 1.0.0 | No                 |

---

## Reporting a Vulnerability

> [!IMPORTANT]
> Please do not report security vulnerabilities via public GitHub issues. Instead, help us keep Phallo safe by reporting security bugs privately.

If you find a security vulnerability, please follow these steps:
1.  Send an email to **283110956+Aswajith-K@users.noreply.github.com** describing the issue.
2.  Provide a description of the potential vulnerability, including steps to reproduce it or a proof-of-concept (PoC) if possible.
3.  We will acknowledge receipt of your email within 48 hours and work with you to analyze and patch the vulnerability.
4.  Once resolved, a public disclosure or release notes update will be published.

Thank you for helping keep this project secure!
